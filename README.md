# Лабораторная работа №3 по линейной алгебре

Проект, посвящённый сингулярному разложению матриц (SVD) и его применению к сжатию данных и изображений.

В работе исследуются усечённое SVD, низкоранговые аппроксимации, норма Фробениуса и представление матрицы в виде суммы компонент ранга 1. Для вычислений и визуализации используются Python, NumPy, SciPy, Matplotlib и средства обработки изображений.
